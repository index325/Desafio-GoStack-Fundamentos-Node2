## :computer: Tech:
- NodeJS
- Typescript
- Express
- ESLint + Prettier
- SOLID

## :running: Run the project:
```shell
  # Clone this repository
  - git clone https://github.com/index325/Desafio-GoStack-Fundamentos-Node2.git
  
  # Enter project directory
  - cd Desafio-GoStack-Fundamentos-Node2
  
  # Install dependencies
  - yarn
  
  # Run the tests:
  - yarn test
  
  # Start the server
  - yarn dev
```
